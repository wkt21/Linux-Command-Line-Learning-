# Linux-Command-Line-Learning

<img width="1168" height="784" alt="IMG_1962" src="https://github.com/user-attachments/assets/1c137aac-bd42-4994-ac68-0392a075b382" />


This repository is a structured Linux command-line learning project covering:

- Creating files and folders.
- Moving and copying files.
- Creating users and groups.
- Managing permissions and ownership.
- Deleting files and directories.
- Downloading and installing software.
- Creating and hiding files and folders.
- RSA and AES encryption using the command line.

## Topics

### 1. Files and folders
- `touch`
- `mkdir`
- `cp`
- `mv`
- `rm`
- `ls`

### 2. Users and groups
- `useradd`
- `usermod`
- `groupadd`
- `gpasswd`
- `id`
- `passwd`

### 3. Permissions
- `chmod`
- `chown`
- `chgrp`
- `umask`
- symbolic and numeric modes

### 4. Downloading and installing
- `curl`
- `wget`
- `apt`
- `dnf`
- `yum`
- `rpm`
- `dpkg`

### 5. Hidden files
- dotfiles
- `ls -a`
- `.secret`
- hidden directories

### 6. Encryption
- `openssl enc` for AES
- `openssl genpkey` and `openssl rsa` for RSA key workflows
- file encryption and decryption examples

## Goal

The goal of this repo is to provide practical, command-line-only examples for Linux beginners and operators.

## Safety note

Commands related to users, permissions, and encryption should be tested in a lab or virtual machine before use on production systems.
